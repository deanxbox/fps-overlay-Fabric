# FPS Overlay (Fabric)

A client-side Minecraft mod designed to display your current frames per second with a clean UI, without cluttering the screen or impacting gameplay performance.

## Features
* **Lightweight:** Minimal impact on game performance.
* **Clean UI:** Displays FPS clearly without unnecessary screen clutter.
* **Client-Side:** Only needs to be installed on your client, not the server.

## Downloads
You can download the official releases of this mod on:
* [Modrinth](https://modrinth.com/project/j0IQ4hjv)
* [CurseForge](https://www.curseforge.com/minecraft/mc-mods/fpsoverlay)

## Installation
This is a Fabric mod. To install it:
1. Ensure you have the [Fabric Loader](https://fabricmc.net/) installed for your version of Minecraft.
2. Download the latest `.jar` file from the releases page, Modrinth, or CurseForge.
3. Place the downloaded `.jar` file into your `.minecraft/mods` folder.
4. Launch the game using the Fabric profile.

## Building from Source
If you want to compile the mod yourself:
1. Clone the repository.
2. Open a terminal in the project directory.
3. Run `./gradlew build` (or `gradlew.bat build` on Windows).
4. The compiled `.jar` will be located in `build/libs/`.
